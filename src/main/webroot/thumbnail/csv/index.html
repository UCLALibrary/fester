CSV Upload

The CSV upload form can be used to upload festerized CSV files for adding thumbnails.

Instructions

To start a batch job, upload a CSV file (like one of those found in the Eureka GitHub repository).


Problems?

It's possible to submit a job that takes longer than the timeout on your browser allows. For really large CSV submissions, you might need to use the Festerize script, which does not have a timeout.

If you run into issues when using this upload form, please feel free to contact us in the #softwaredev-services Slack channel. We will be glad to troubleshoot any problems that you may encounter. Thanks!
